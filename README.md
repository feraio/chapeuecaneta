# CHAPÉU&CANETA® - Landing Page

Landing page institucional e vitrine de produtos da marca **CHAPÉU&CANETA®**.
O projeto foca em alta performance, SEO otimizado e uma estética Dark/Bold alinhada ao posicionamento da marca, Futebol é Arte.

**URL do Projeto:** [https://chapeuecaneta.com.br/](https://chapeuecaneta.com.br/)

## Tecnologias Utilizadas

- **HTML5 Semântico** (SEO Friendly)
- **CSS3 Moderno** (Flexbox, CSS Variables, Text-Wrap Balance)
- **Swiper.js** (Carrossel touch-friendly)
- **Google Fonts** & Web Fonts otimizadas

## Design System (Guia de Estilo)

### Cores Principais
| Cor | Hex | Uso |
| :--- | :--- | :--- |
| **Preto Fosco** | `#111111` | Background Geral / Gradientes |
| **Amarelo Neon** | `#FFC700` | Botões / Destaques / Seleção |
| **Branco Off** | `#FEFEFE` | Textos / Títulos |

### Tipografia
- **Títulos (Display):** `Boldonse` (Estilo Poster/Condensada)
- **Textos (Body):** `Inter` ou `Helvetica Neue` (Legibilidade)

### Assets & Imagens
- **Hero Desktop:** 1920x1080px (WebP)
- **Hero Mobile:** 1080x1920px (WebP) - *Mobile First*
- **Produtos:** 600x600px (Quadrado perfeito 1:1)

## Performance & Otimizações

- **Preload:** Carregamento prioritário da imagem Hero (LCP otimizado).
- **Lazy Loading:** Imagens do carrossel carregam apenas ao rolar a página.
- **CLS (Layout Shift):** Dimensões de largura/altura definidas no HTML para evitar "pulos".
- **SEO:** Meta tags completas (OpenGraph, Twitter Cards) e JSON-LD Schema para organização.
